# Academic Aid – FOT



A community-driven study platform built for students of the Faculty of Technology (FOT).

## What it does

- Browse study materials, notes, and past papers organized by department (CSE, ECE, EE)
- View syllabi and resources for each branch
- Ask questions and get help from fellow students
- Contribute your own notes to help others out

## Tech Stack

- HTML, CSS, JavaScript (vanilla, no frameworks)
- React
- Firebase (Firestore for data, Firebase Storage for files)
